## ralfpack

A tool for Creating, Signing, Verifying and Inspecting RALF (RDK Application Layer Format) package files.

### Options

```
  -v, --verbose...  Increase output verbosity, can be used multiple times
  -h, --help        Print help
  -V, --version     Print version
```

### Commands

```
  create   Create a new package
  convert  Convert a EntOS widget to RALF package format
  sign     Sign or resign an existing package
  verify   Verify a package signature
  info     Display information about a package
  help     Print this message or the help of the given subcommand(s)
```
